profesi cita-cita sebagai web developper
Profesi Keterangan
Front-End Developper Fokus pada pengembangan bagian depan situs web, termasuk tata letak, desain, dan interaksi pengguna, menggunakan HTML, CSS, dan JavaScript.
Back-End Developper Bertugas mengembangkan sisi server dari situs web, memproses logika bisnis, dan mengelola database, menggunakan bahasa pemrograman seperti Python, Ruby, atau Node.js.
UI/UX Desainer profesi yang fokus pada desain antarmuka pengguna (UI) dan pengalaman pengguna (UX) dalam pengembangan aplikasi, situs web, atau produk digital lainnya.
Quality Assurance (QA) menguji perangkat lunak atau situs web untuk menemukan dan melaporkan bug, memastikan bahwa produk akhir memenuhi standar kualitas yang ditetapkan sebelum dirilis ke pasar atau digunakan oleh pengguna akhir.
